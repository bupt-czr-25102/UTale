# UTale
北邮雏雁项目：邮记UTale
项目概述

## 一、项目简介

**项目名称**：邮记UTale
**项目类型**：微信小程序应用
**项目目标**：开发一款集AI日记、AI视频、日程管理和社区功能于一体的综合性小程序，为用户提供智能、便捷的个人管理和社交服务。

## 二、功能模块

### 1. AI日记模块
- **核心功能**：智能日记生成、日记管理、AI辅助写作
- **技术实现**：前端微信小程序页面 + 后端Flask API + DeepSeek AI服务
- **文件位置**：`frontend/pages/diary/` 和 `backend/modules/diary/`

### 2. AI视频模块
- **核心功能**：AI视频生成、视频管理、视频编辑
- **技术实现**：前端微信小程序页面 + 后端Flask API + DeepSeek AI服务
- **文件位置**：`frontend/pages/video/` 和 `backend/modules/video/`

### 3. 日程管理模块
- **核心功能**：日历视图、日程管理、提醒功能
- **技术实现**：前端微信小程序页面 + 后端Flask API
- **文件位置**：`frontend/pages/schedule/` 和 `backend/modules/schedule/`

### 4. 社区模块
- **核心功能**：帖子发布、评论互动、社区交流
- **技术实现**：前端微信小程序页面 + 后端Flask API
- **文件位置**：`frontend/pages/community/` 和 `backend/modules/community/`

## 七、项目特色

1. **智能AI集成**：利用DeepSeek AI服务，实现智能日记和视频生成功能
2. **模块化设计**：采用前后端分离架构，模块间低耦合，便于开发和维护
3. **清新界面风格**：采用淡蓝色系的清新设计风格，提供良好的用户体验
4. **全栈开发体验**：团队成员每人负责一个完整模块的前后端开发，获得全栈开发经验
5. **学习型项目**：项目以学习为目的，团队成员边学习边开发，共同成长
